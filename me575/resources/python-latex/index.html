Choose Your Own Adventure

Python and Matlab are both excellent choices. If you don’t feel reasonably comfortable with your programming ability I suggest you stick with what you know. I assume you’ve been exposed to Matlab, so I won’t elaborate on its advantages, but do brief discuss some of the advantageous of Python below.

LaTeX is widely used in scientific writing and if you are doing graduate work you will likely need to learn it at some point (but check with your advisor, I know that a few research groups prefer Word). To be clear, there is nothing wrong with using Matlab for your code development, and Microsoft Word (or an equivalent) for your report preparation. These are just tools, what’s more important is learning the concepts and doing good work.

Python

Here is a brief Python primer I prepared for another venue. It is the first half that is most relevant for this discussion. The more advanced topics in the second half: object oriented programming, and connecting with Fortran/C won’t be necessary in this class. For a broader perspective, see the post I wrote on scientific programming languages. Tons of Python tutorials exist online, as do good IDEs that can support autocompletion, setting breakpoints, etc.

LaTeX

Unless you already have a workflow you are comfortable with, I recommend you use Overleaf. Think of it as Google Docs for LaTeX. There isn’t anything to install, and multiple people can work collaboratively (nice for your project). It also provides a lot of starter templates. Many LaTeX tutorials exist online. You might find some of this post I wrote on Writing Scientific Papers useful. Below is a very simple starter example:

\documentclass{article}

\usepackage[margin=1in]{geometry}
\usepackage{amsmath}
\usepackage[colorlinks,bookmarks,bookmarksnumbered,allcolors=blue]{hyperref}
\usepackage{graphicx}
\usepackage{subfig}
\usepackage{booktabs}
\usepackage[capitalise]{cleveref}

\begin{document}

\author{Your Name}
\title{Title of Report}
\maketitle

\section{A Section Header}

Text

\end{document}
