HW 0: Getting Started

due 9/13/2017 before midnight via Learning Suite 10 possible points

Overview: This is not a regular-length homework. Just a small assignment to get us started while we get acclimated during this first week.

  1. On Friday you will give a 5-minute presentation on some person or some event from aerospace history. You will work in groups of 4 (a few will have 5). Sign up on this Google Doc: []. I’ve provided a few ideas, but feel free to use any of your own. Each person in the group should participate in the presentation. If you want to use slides you need to email them to me by 9AM Friday so I can put them all together. You don’t have to use slides if you don’t want to.

  2. Review/learn background concepts. Below I list some fundamental background that you’ll need for this class.

    • Read Chapter 2
    • Review some of the basic concepts from ME EN 312 (as needed): pressure, density, viscosity, ideal gas law, incompressible vs compressible, static vs total pressure, Bernoulli’s equation, conservation of mass/momentum/energy, streamlines, dimensional analysis (i.e., nondimensional numbers).
    • Review some of the basic concepts from ME EN 372 or CE EN 203 (as needed): distributed loads, bending stress, bi-axial bending, torsion, combined loads (i.e., bending, axial, torsion).
